Installation

BlueSentry is built with Python 3.8+. It relies on bleak for Bluetooth access and rich for the UI.

Quick Install

git clone https://github.com/your-repo/bluesentry.git
cd bluesentry
pip install .

Note: On Linux, you usually need sudo to access the Bluetooth adapter.

Usage Guide

1. The Scanner

The primary tool for area surveillance.

# Standard 20-second scan
sudo bluesentry

# Scan for 1 minute and save to a custom file
sudo bluesentry --duration 60 --output myscan.csv

# Passive Mode (No UI, background logging)
sudo bluesentry --passive

2. The Tracker (Bloodhound)

Used to physically locate a specific device.

# Replace with your target's MAC address
sudo python3 tracker.py AA:BB:CC:11:22:33
  • -40 dBm: Very Close (< 1m)
  • -70 dBm: Nearby (Same room)
  • -90 dBm: Weak Signal (Limit of range)

Troubleshooting

"Bluetooth Adapter Not Found"

Ensure your Bluetooth is turned on and not blocked by rfkill.

sudo rfkill unblock bluetooth
sudo systemctl start bluetooth
